# 보안 정책
Claude Code를 안전하게 유지하는 데 도움을 주셔서 감사합니다!

## 보안 이슈 보고

시스템과 사용자 데이터의 보안은 Anthropic의 최우선 사항입니다. 잠재적인 취약점을 식별하고 보고하는 선의의 보안 연구원들의 노력을 소중히 여깁니다.

보안 프로그램은 HackerOne에서 관리되며, 이 기능에서 검증된 취약점은 [제출 양식](https://hackerone.com/anthropic-vdp/reports/new?type=team&report_type=vulnerability)을 통해 보고해 주시기 바랍니다.

## 취약성 공개 프로그램

취약성 프로그램 가이드라인은 [HackerOne 프로그램 페이지](https://hackerone.com/anthropic-vdp)에 정의되어 있습니다.
